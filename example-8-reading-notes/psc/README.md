#### 强引用和弱引用是什么, 有什么用

强引用和弱引用, 发生于链接阶段, 解决同名符号的问题.  链接阶段,
对强符号和弱符号的引用称为强引用和弱引用. 函数和初始化的全局变量的引用为强引用.
强引用的符号不可重复定义, 否则报错, 重复定义的场景会出现在两个不同的对象文件
`.o` 扩展名的文件, 强引用和弱引用就是用于解决链接这些对象文件时,
出现同名符号的问题.

弱引用方法还可以用于判断当前的程序是链接到单线程的 Glibc 还是多线程的 Glibc.
